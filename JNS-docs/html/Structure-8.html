8. Additional Comments

As mentioned before already, you should not really have to do all this setup work, particularly if you want to simulate a rather large network (Internet, anyone?). Here are some suggestions on how to make your life easier:

  • ALWAYS attach all your elements to the Simulator. Your simulation might work without doing it but you will not be able to trace anything.
  • Use loops if you have many nodes.
  • Write a class that will produce "shrink-wrapped" Nodes. A bit like a mass computer retailer: Complete with an interface and routing table. You could then produce a node with one line of code.
  • Abstract this idea further to produce whole networks shrink-wrapped.
  • Alternatively, forget about all this. Download one of the many good topology generators available on the web and either write a script or modify its code so it directly generates JNS code.
