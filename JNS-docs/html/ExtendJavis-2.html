2. How to hook in your own handlers

If you have read the previous section, this should be fairly obvious. You created your own subclass of "JavisHandler". When you're finished doing that, you can register this class with the JavisTrace class. You have to do this after creation of a JavisTrace object, but before any output has taken place.

You can achieve this by calling JavisTrace.registerHandler, pass as arguments the name of the event your class handles and an instance of the class.

If you don't have a clue how to write the actual handler class, the best thing to do is to look at one of the hard-coded ones, e.g. NodeHandler. Make sure you do not look at the one with the longest and most complicated bit of code first (e.g. PacketHandler). If you do have a clue, you're done, enjoy it.
